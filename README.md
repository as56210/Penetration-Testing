### Penetration Testing

The class was instructed to scan the given network, find the vulnerable machine on the network, enumerate the machine, use Nessus to find the machine's vulnerabilities, and use Metasploit to exploit the machine. The class was also tasked with downloading and reading files on the target machine, and writing a recommendation on how to potentially fix the vulnerability.

# Tools Used
- Kali Linux Virtual Machine
- Network Mapper
- Nessus
- Metasploit
